## Switch statement
```java
    String color = in.nextLine();
    
    switch (color)
    {
      case "green": System.out.println("Green"); break;
      case "red": System.out.println("Red"); break;
      case "orange": System.out.println("Orange"); break;
      default: System.out.println("Nothing matched!"); break;
    }
```
